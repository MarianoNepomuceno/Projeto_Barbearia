# Sistema de Gestão para Barbearia

> Projeto desenvolvido para a disciplina de Prática Profissional Supervisionada.

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar um sistema de gerenciamento para barbearias, oferecendo uma solução prática e organizada para o controle de clientes, serviços e agendamentos.

A proposta surgiu a partir da necessidade de modernizar processos que normalmente são realizados manualmente, proporcionando mais eficiência no atendimento e melhor organização das informações.

O sistema foi pensado para atender uma barbearia de bairro localizada na região onde o desenvolvedor reside, buscando oferecer uma solução acessível, funcional e adaptada às necessidades do comércio local.

---

## Objetivos do Sistema

- Facilitar o gerenciamento da barbearia;
- Organizar informações de clientes e serviços;
- Melhorar o controle de atendimentos e agendamentos;
- Reduzir erros em processos manuais;
- Oferecer uma interface simples e intuitiva.

---

## Funcionalidades

### Gestão de Clientes
- Cadastro de clientes;
- Consulta de informações;
- Atualização de dados;
- Remoção de registros.

### Gestão de Serviços
- Cadastro de serviços oferecidos;
- Controle de preços;
- Organização dos atendimentos.

### Agendamentos
- Marcação de horários;
- Controle de atendimentos;
- Visualização de agendamentos.

### Controle Administrativo
- Organização geral do sistema;
- Melhor gerenciamento das informações;
- Maior praticidade no dia a dia da barbearia.

1. Faça o download ou clone este repositório;
2. Abra a pasta do projeto;
3. Execute os arquivos `index.html` e `admin.html` em seu navegador.
